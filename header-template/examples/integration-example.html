Header Template Integration Example

This example demonstrates how to integrate the header template into your pages.

Features Demonstrated:

  • ✅ Responsive navigation
  • ✅ Mobile menu functionality
  • ✅ Scroll progress indicator
  • ✅ Header state changes on scroll
  • ✅ Accessibility features (ARIA labels, keyboard navigation)
  • ✅ Smooth scrolling to sections

Integration Methods:

Method 1: Server-Side Include

<!-- PHP -->
<?php include 'header-template/components/header.html'; ?>

<!-- Apache SSI -->
<!--#include file="header-template/components/header.html" -->

Method 2: JavaScript Dynamic Loading

// Load header dynamically
fetch('header-template/components/header.html')
    .then(response => response.text())
    .then(html => {
        document.getElementById('header-container').innerHTML = html;
    });

Method 3: Build Process Integration

// Webpack HTML loader or similar
import headerTemplate from './header-template/components/header.html';
